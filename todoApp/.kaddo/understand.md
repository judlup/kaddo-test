## Copy/Paste Instructions

Start with **roadmap-agent**:

1. Open your preferred LLM chat (Claude, ChatGPT, Cursor, Copilot, Windsurf…).
2. Paste the context pack: `.kaddo/context-pack.md`
3. Paste the agent prompt: `knowledge/agents/roadmap-agent.md`
4. Ask the LLM to produce: `knowledge/delivery/roadmap.md`
5. Save the result in: `knowledge/delivery/roadmap.md`

## Next Steps

After saving `knowledge/delivery/roadmap.md`, continue with **architecture-agent** (feed it the context pack plus the artifacts you already produced). Re-run `kaddo understand` any time to see this plan again.
